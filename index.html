Recycle API

Base URL: https://dawande.000webhostapp.com/api/

1.  Get Recycle Info (GET) /info.php

URL params:

Name

Obligatory

Description

id

-

[globalId] of recycle info record

barcode

-

Barcode value of the scanned product

barcodeType

-

Scanned barcode format. One of BarcodeType emun constants (see below)

Response JSON Body:

{

    /**

     * Records that fully fits the request

     * @type {Recycle[]}

     */

    "fullMatch": [],

    /**

     * Records that partially fits the request

     * @type {Recycle[]}

     */

    "partialMatch": [],

    /**

     * Null if on error occurred. String with an error description otherwise

     * @type {null|string}

     */

    "error": null

}

Note:

·          As all params are not obligatory, it doesn’t mean that request can be empty. Empty request (/info.php?) returns error response;

·          If you specify barcode, please specify the barcodeType also;

·          id param returns only one object in fullMatch array.

 

2.  User sign out (POST) /registration.php

Request body (JSON of formatted):

Name

Obligatory

Type

Description

login

+

Str, max 255

User login.

password

+

Str, max 50

User password.

mac

+

Str

Device MAC. (Хз пока, зачем, но пусть будет)

Response JSON Body:

{

    /**

     * User login

     * @type {string}

     */

    "login": "",

    /**

     * User unique identifier

     * @type {int}

     */

    "userId": 0,

    /**

     * Null if no error occurred. String with an error description otherwise

     * @type {null|string}

     */

    "error": null

}

Note:

·        UserId less than 1 is invalid numbers and used to indicate error.

 

3.  Offer Recycle Info Record (POST) /offer.php

Request body (JSON of formatted):

Name

Obligatory

Type

Description

userId

+

Int

User login. Only registered users can offer records.

login

+

Str

User login. Only registered users can offer records.

globalId

-

str|null

Product globalId. Only if offer edit to existing record.

name

-

str|null

Product name

barcode

+

Str

Scanned barcode

barcodeType

+

Str

Scanned barcode format (see BarcodeType below)

productInfo

-

str|null

Info about product.

utilizeInfo

-

str|null

Info about product utilization.

image

-

str|null

Bitmap encoded with Base64

Response JSON Body:

{

    /**

     * User login. Equals to login from request

     * @type {string}

     */

    "login": "",

    /**

     * Null if no error occurred. String with an error description otherwise

     * @type {null|string}

     */

    "error": null

}

Note:

·        At least 1 param must be present and non-null. Otherwise error will be thrown.
